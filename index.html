🎧 Audiovisualizer

This app transforms music into color using audio frequency analysis. It splits any .wav file into 1000 equal parts, detects the dominant frequency in each, maps that to a visible color (based on human perception), and generates a beautiful RGB-based visual representation of the sound.

🛠 Features

  • Analyzes local .wav files
  • Maps dominant frequency to RGB color using logarithmic scaling
  • Renders a color strip representing the audio's spectrum
  • No data collection — fully offline
